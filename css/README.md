# CSS Styles
Useful styles


### Button styling with fading hover effect using transition-duration
```css
.button {
	transition-duration: 0.4s
}

.button:hover {
	background-color: blue;
	color: white
}
```css


### Button styling with fading in using opacity
```css
.button {
	opacity: 0.6;
}

.button:hover {
	opacity: 1
}
.
