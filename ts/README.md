# Typescript
https://www.typescriptlang.org/
JS with types (and more)


## Getting started
### Install typescript
```
pnpm add typescript -D
```

### Run Typescript compiler
```
npx tsc
```

### Install globally
```
npm install -g typescript
```

### Install ts-node
```
npm install -g ts-node
```

### Install node.js types for ts
```
pnpm add @types/node -D
```

### Run ts script
```
ts-node main.ts
```
